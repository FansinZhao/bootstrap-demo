图片嵌套

第一点: media 类

你必须使用class="media",同时要指定你需要的对齐方式.

第二点

图像要使用img标签,使用data-src属性指定图片大小,同时要制定width和height

第三点

img标签可以嵌套使用!这个是bootstrap的一大特色,你可以随意组合,而不用太多关注他们如何契合

第四点

图片标题使用media-heading来制定

第五点

可以通过改变标签的位置随意组合布局
